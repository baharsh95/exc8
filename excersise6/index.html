HTML (Hyper Text Markup Language) in Kombination mit CSS und JavaScript sind die Standard-Tools zum Entwickeln von WebApps und browserbasierten Oberflächen für alle möglichen Anwendungen. In diesem Training erhalten Sie kompakt in 3 Tagen das wichtigste Know-how und können sofort mit dem Programmieren starten

  1. html
  2. css
  3. javascript

Dieses Seminar bietet eine Einführung in HTML5, CSS3 und JavaScript. Die Kursinhalte werden anhand zahlreicher Demos veranschaulicht und in Übungen gefestigt und wiederholt. Als Entwicklungsumgebung wird im Training Visual Studio Code verwendet, das auf allen Plattformen funktioniert.

  • Webseiten erstellen mit HTML, CSS und Javascript
  • HTML-Elemente zum Strukturieren von Webseiten
  • Layout und Design mit CSS
  • Einführung in Javascript und DOM
